Link Page Editor

自動保存

使い方 & 公開手順

1

プロフィールを編集する

左のサイドバーで 名前・肩書き・自己紹介文・アバター画像URL を入力します。

アバターは https:// から始まる画像URL、または同じフォルダにある画像ファイル名（例：photo.jpg）が使えます。SNSのプロフィール画像のURLをそのまま貼ってもOKです。

2

SNSリンクを設定する

「SNSリンク」セクションで各リンクの タイトル・URL・アイコン を入力します。

アイコン欄には 🐦 🔗 📸 などの絵文字も使えます。
↑↓ボタンで並び替え、✕で削除できます。

3

右プレビューで確認する

編集内容はリアルタイムで右側に反映されます。「スマホ」ボタンでスマートフォン表示も確認できます。

内容はブラウザに自動保存されるので、途中でページを閉じても大丈夫です。

4

「HTMLをダウンロード」を押す

links.html というファイルがダウンロードされます。これがあなたのリンクページの完成ファイルです。

公開方法を選ぶ

かんたん

Netlify Drop — ドラッグ＆ドロップで即公開

  1. app.netlify.com/drop をブラウザで開く
  2. ダウンロードした links.html をページ中央にドラッグ＆ドロップ
  3. 数秒で https://ランダム名.netlify.app のURLが発行される
  4. そのURLを名刺やSNSに貼り付けて完了！

※ アカウント登録不要。ただし無料プランのURLは定期的に変わる場合があります。

本格

GitHub Pages — 自分のURLで公開

  1. github.com にサインアップ・ログイン
  2. 右上「＋」→「New repository」でリポジトリを作成
    名前を あなたのユーザー名.github.io にする
  3. 「Add file」→「Upload files」で links.html をアップロード
  4. Settings → Pages → Source を main ブランチに設定
  5. https://あなたのユーザー名.github.io/links.html でアクセス可能になる（反映まで数分）
💡 ワークショップTips アバター画像は事前にGoogle Drive等にアップしておいて「共有リンクを取得」→「誰でも閲覧可」にしておくと、URLとして使いやすいです。
Netlify Drop は会場のWi-Fiがあれば即URLを発行できるので、名刺QRコード用途なら一番手軽です。
