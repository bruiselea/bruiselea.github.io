# Phone as Mouse

スマホをマウス&キーボードとして使うFlutterアプリ + Pythonサーバー

## 機能

- **ジェスチャーモード**: 画面をドラッグしてマウス移動(タッチパッド風)
- **加速度センサーモード**: スマホを傾けてマウス移動
- **フリックキーボード**: 日本語入力対応(予定)

## セットアップ

### 1. PC側サーバーの起動

```bash
# 依存関係のインストール
pip3 install -r requirements.txt

# サーバー起動
python3 pc_receiver.py
```

サーバーが起動すると、接続用のIPアドレスが表示されます。

### 2. スマホアプリの起動

```bash
flutter run
```

アプリ起動時に接続ダイアログが表示されるので、PC側サーバーのIPアドレスを入力してください。

## 使い方

### ジェスチャーモード(デフォルト)
- **ドラッグ**: マウス移動
- **タップ**: 左クリック
- **長押し**: 右クリック
- **ダブルタップ**: ダブルクリック

### 加速度センサーモード
1. "Switch to Accelerometer"ボタンをタップ
2. "Start Tracking"ボタンをタップ
3. スマホを傾けるとカーソルが動く

## トラブルシューティング

### 接続できない場合
- PCとスマホが同じWiFiネットワークに接続されているか確認
- ファイアウォールでポート8765が開いているか確認
- IPアドレスが正しいか確認

### PCのIPアドレスの確認方法
- **macOS**: システム設定 > ネットワーク
- **Windows**: `ipconfig`
- **Linux**: `ip addr`
