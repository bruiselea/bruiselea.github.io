# 佐藤夏輝 - ポートフォリオサイト

東北大学で材料工学を専攻し、3Dプリンティングとプログラミングに情熱を注ぐ学生のポートフォリオサイトです。

## 🚀 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: モダンなデザインとレスポンシブレイアウト
- **JavaScript (ES6+)**: インタラクティブな機能
- **Font Awesome**: アイコンライブラリ
- **GitHub Pages**: ホスティング

## ✨ 機能

- **レスポンシブデザイン**: モバイル、タブレット、デスクトップに対応
- **スムーズスクロール**: ページ内ナビゲーション
- **アニメーション**: スクロール時のフェードインアニメーション
- **お問い合わせフォーム**: バリデーション機能付き
- **モバイルメニュー**: ハンバーガーメニュー対応
- **SEO最適化**: メタタグとセマンティックHTML

## 📁 ファイル構成

```
bruiselea.github.io/

```

## 🔧 セットアップ

### GitHub Pagesでの公開

1. このリポジトリをGitHubアカウントにフォークまたはクローン
2. リポジトリ設定の「Pages」セクションに移動
3. ソースを「Deploy from a branch」に設定
4. ブランチを「main」に設定
5. 自動的にサイトが公開されます

### ローカル開発

```bash
# リポジトリをクローン
git clone https://github.com/bruiselea/bruiselea.github.io.git

# ディレクトリに移動
cd bruiselea.github.io

# ローカルサーバーを起動（Python 3の場合）
python -m http.server 8000

# ブラウザでアクセス
# http://localhost:8000
```

## 🎨 カスタマイズ

### 基本情報の変更

`index.html` の以下の部分を編集してください：

- **名前**: `<h1 class="hero-title">NatsukiSato</h1>`
- **肩書き**: `<p class="hero-subtitle">材料工学 × 3Dプリンティング × AI</p>`
- **自己紹介文**: `<p class="hero-description">...</p>`
- **連絡先情報**: お問い合わせセクションのリンク

### スタイルの変更

`styles.css` で以下の要素をカスタマイズできます：

- **カラーパレット**: CSS変数を使用した色の変更
- **フォント**: font-familyの変更
- **レイアウト**: グリッドやフレックスボックスの調整

### 機能の追加

`script.js` に新しい機能を追加できます：

- **アニメーション**: 新しいエフェクトの追加
- **フォーム**: より高度なバリデーション
- **API連携**: 外部サービスとの連携
